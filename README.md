# 🧠 MindSarthi AI – Mental Health Support Platform

MindSarthi AI is an intelligent mental health support and assistance platform that provides emotional support, AI-based chatbot interaction, counseling services, and psychology-based learning resources.

---

## 🚀 Features

- 🤖 AI Chatbot (Gemini API based)
- 🧠 Mood Detection & Sentiment Analysis
- 📊 User Dashboard with Mood Trends
- 📚 Psychology Courses with Quizzes
- 📄 Auto Certificate Generation (PDF)
- 📧 Email Notifications (Registration, Contact, Counseling)
- 🏥 Doctor & Hospital Finder (Location-based)
- 👩‍⚕️ Counseling Session Request System

---

## 🛠️ Tech Stack

**Frontend:**
- HTML, CSS, JavaScript

**Backend:**
- Python (Flask)

**Database:**
- JSON (for prototype)

**APIs Used:**
- Google Gemini API (AI Chatbot)
- OpenStreetMap (Doctors & Hospitals Search)

**Other Tools:**
- SMTP (Email Service)
- ReportLab (PDF Generation)
